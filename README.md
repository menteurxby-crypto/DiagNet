# DiagNet - Network Diagnostic Tool
## App Introduction
DiagNet is a powerful network diagnostic tool designed for both casual users and IT professionals. It provides advanced Ping testing, batch Ping, and network monitoring capabilities to help you easily monitor and troubleshoot network issues.

## Usage Guide
### Ping Test
1. Open the DiagNet app
2. Enter the IP address or domain name to test in the target input field
3. Tap the start button to begin the Ping test
4. View real-time results and statistics
### Batch Ping
1. Enter multiple IP addresses or domain names in the target input field, one per line
2. Tap the start button to begin batch Ping testing
3. View real-time results for all targets
4. Save or export test results
